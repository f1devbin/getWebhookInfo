# 🛠 Advanced Telegram Bot Configuration Manager

Профессиональный веб-интерфейс (Single-page Web Panel) на PHP для мгновенной диагностики, администрирования и тонкой настройки параметров Telegram Bot API без использования тяжелых фреймворков.

## 🔥 Возможности панели
* **Управление Webhook:** Быстрая регистрация, изменение или полное удаление URL-адреса обработчика на серверах Telegram.
* **Интерактивный селектор событий (`allowed_updates`):** Удобная сетка чекбоксов со встроенными подробными подсказками для каждого типа обновлений (`message`, `callback_query`, `chat_member` и др.) вместо чтения громоздкой документации.
* **Синхронизация и очистка команд бота:** Возможность в один клик обновить визуальное меню команд в Telegram-клиентах пользователей или полностью стереть все активные команды (`deleteMyCommands`).
* **Дефолтные права администратора:** Настройка базовых разрешений для автоматического запроса прав при добавлении бота в супергруппы и каналы.
* **Автономность:** Скрипт работает в один файл, не требует базы данных SQL и собирает базовую статистику обращений в локальный JSON-файл.

## 🚀 Быстрый старт
1. Склонируйте репозиторий или перенесите файл `index.php` в веб-директорию вашего сервера (Nginx / Apache с поддержкой PHP 8.x).
2. Откройте страницу в браузере.
3. Введите HTTP API Токен вашего бота, полученный от `@BotFather`, для инициализации сессии и чтения текущих настроек.

## 🔒 Безопасность
* Скрипт отправляет запросы напрямую к официальному шлюзу `api.telegram.org` по защищенному протоколу HTTPS.
* Токены не сохраняются на сервере в открытом виде и используются только в рамках текущей сессии/запросов.
